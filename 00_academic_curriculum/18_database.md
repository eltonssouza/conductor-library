## Database System

| Field | Value |
|-------|-------|
| Authors | Silberschatz, A., Korth, H., Sudarshan, S. |
| Publication Year | 2020 |
| Edition | 7th edition. |
| ISBN | 9788595157330 |

### Description

Everything about database systems: from the fundamentals to the newest concepts. With practical and theoretical questions, this book presents the application, management, and implementation of databases and includes recent topics in the field, such as Big Data and blockchain. As a differentiator, Database System presents concepts and algorithms in a general environment, not tied to a specific data system, which makes it flexible to apply the theory to diverse scenarios. In addition, it offers a variety of exercises at different levels of complexity and learning-support tools.This edition of Database System is necessary because it brings:Extensive coverage of Big Data systems, from the user's point of view, as well as an internal perspective;Updates (in every chapter) on the internal aspects of databases, to reflect state-of-the-art technology, such as solid-state drives (SSD), main-memory databases, multi-core systems, and parallel and distributed databases;Advanced coverage of semi-structured data management using JSON, RDF, and SPARQL;Updated coverage of temporal data, data analysis, and advanced indexing techniques, such as write-optimized indexes;Reorganization and updating of the chapters to better support courses, with a significant practical component, including the use of state-of-the-art tools for application development and Big Data systems, such as Apache Hadoop and Spark;A new chapter on Blockchain Databases, which introduces the technology and its increasingly important role in enterprise applications.

## Database Systems

| Field | Value |
|-------|-------|
| Authors | Elsmari, R., Navathe, Shamkant B. |
| Publication Year | 2019 |
| Edition | 7th edition. |
| ISBN | 9788543025001 |

### Description

Database Systems presents, in a deep and up-to-date way, the most important aspects of database systems and applications, as well as related technologies. This reference work introduces the fundamental concepts needed to design, use, and implement database systems, emphasizing the fundamentals of modeling, design, languages, and models provided by management systems and implementation techniques. In this edition, new chapters were introduced on recent advances in database systems and Big Data processing, an introduction to NOSQL databases, and technologies for big data processing. In addition, new examples were included, and the content on query processing and optimization and its techniques, strategies, and algorithms for processing was expanded, and new SQL constructs were added. With updated content, this work is ideal for students of network analysis, systems analysis, data processing, and computer engineering who want to master the subject and stand out in the field.

## Database Management Systems

| Field | Value |
|-------|-------|
| Authors | Ramakrishnan, R. |
| Publication Year | 2008 |
| Edition | 3rd edition. |
| ISBN | 9788577260270 |

### Description

This book presents a clear and up-to-date approach to the fundamentals of database systems. Instructors can choose to emphasize database applications or systems. The book brings many examples and applications, including SQL and Oracle, and applications for the Internet. In addition, current standards such as JDBC and XML, and three-tier application development, are introduced.
